# Devang Bhardwaj - Portfolio Website

A modern, responsive portfolio website showcasing projects, achievements, and skills.

## 🚀 Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling, hover effects, and dynamic content
- **Performance Optimized**: Fast loading with optimized assets
- **Accessible**: Built with accessibility best practices

## 📋 Sections

- **Home**: Hero section with introduction and call-to-action
- **About**: Personal information and background
- **Resume**: Education, skills, and experience
- **Projects**: Showcase of development projects
- **Achievements**: Notable accomplishments and awards
- **Certificates**: Professional certifications and courses
- **Contact**: Contact form and social media links

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive functionality and animations
- **Font Awesome**: Icons and visual elements
- **Google Fonts**: Typography (Inter font family)

## 🎨 Design Features

- Gradient backgrounds and modern color scheme
- Smooth scroll animations and transitions
- Mobile-first responsive design
- Interactive navigation with active states
- Professional card-based layouts
- Optimized typography and spacing

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: 320px - 767px

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information
4. Deploy to your preferred hosting platform

## 📝 Customization

To personalize the website:

1. **Update Personal Information**: Edit the content in `index.html`
2. **Add Your Projects**: Replace placeholder projects with your actual work
3. **Update Contact Info**: Add your email and social media links
4. **Add Your Photo**: Replace the profile placeholder with your image
5. **Modify Colors**: Customize the CSS color scheme in `styles/main.css`

## 🌐 Deployment

This website can be easily deployed to:

- **GitHub Pages**: Push to a GitHub repository and enable Pages
- **Netlify**: Connect your repository for automatic deployments
- **Vercel**: Import your project for instant deployment
- **Any Static Hosting**: Upload files to any web hosting service
